# VisionVoice

VisionVoice is an AI-powered Android application developed to assist visually impaired individuals by providing real-time voice guidance and intelligent computer vision features. The application uses deep learning and speech technologies to recognize objects, faces, signboards, and facial expressions, helping users navigate their surroundings more independently.

---

## Features

- 👤 Face Recognition
- 😊 Facial Expression Detection
- 🚶 Person Detection
- 📦 Object Detection
- 🛣️ Sign Board Detection
- 🗣️ Voice Guidance using Text-to-Speech
- 🗺️ Location and Map Support
- 👨‍👩‍👧 Family Member Login and Registration
- 📷 Image Upload and Recognition

## Installation

1. Clone the repository

```bash
git clone https://github.com/AnnreaLijo/Visionvoice_app.git
```

2. Open the project in Android Studio.

3. Sync Gradle files.

4. Add your own API keys where required (do not use the placeholder values).

5. Connect an Android device or emulator.

6. Build and run the application.
